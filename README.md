# k8s

This repository is created for learning Kubernetes (K8s) through hands-on practice using Kubernetes manifest files.

The primary goal of this repository is to improve my skills in Kubernetes, gain a deeper understanding of its core concepts, and become more proficient by working directly with YAML-based configuration files.

---

## Purpose

- Learn Kubernetes in a practical and structured manner  
- Improve understanding of Kubernetes architecture and components  
- Practice writing clean, correct, and readable manifest files  
- Develop better troubleshooting and operational skills  
- Build a strong foundation for real-world Kubernetes environments  

---

## Repository Content

This repository contains Kubernetes manifest files related to various Kubernetes resources and concepts, including but not limited to:

- Pods  
- Deployments  
- ReplicaSets  
- Services  
- ConfigMaps  
- Secrets  
- Volumes  
- Liveness, Readiness, and Startup Probes  
- Jobs and CronJobs  
- Scheduling and resource management  

All examples are intended strictly for learning and experimentation purposes.
